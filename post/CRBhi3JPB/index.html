vue移动端适配

2019-11-13

安装插件

npm i lib-flexible --save        // 载lib-flexible
npm install px2rem-loader        // 安装px2rem-loader

postcss.config.js配置

module.exports = {
  plugins: {
    autoprefixer: {},
    "postcss-pxtorem": {
      "rootValue": 32,
      "propList": ["*"]
    }
  }
}

如果引入了第三方插件的话

插件安装

 npm install postcss-px2rem-exclude --save

postcss.config.js配置

module.exports = {
  plugins: {
    autoprefixer: {},
    'postcss-px2rem-exclude': {
      remUnit: 32,
      exclude: /node_modules|folder_name/i
    }
  }
}
